Description du jeu :  
  Le jeu Morpion, également connu sous le nom de Tic-Tac-Toe, est un jeu de société simple où deux joueurs s'affrontent pour aligner trois symboles identiques horizontalement, verticalement ou en diagonale sur une grille de 3x3 cases.  
  Le premier joueur utilise le symbole 'X', tandis que le second utilise 'O'. Si toutes les cases sont remplies sans qu'un joueur aligne trois symboles, la partie se termine sur une égalité.  

Installation :  
  Décompressez l'archive du jeu, ce qui crée un répertoire mini_projet.  
  Dans un terminal Linux/Unix, placez-vous dans le répertoire du jeu  
  et utilisez la commande `make` pour compiler le programme.  


Comment jouer :  
  Lancer le programme morpion_jeu.  
  Sous Linux, utilisez la commande `./morpion_jeu`.  
  Au lancement, le jeu affiche un menu qui invite les utilisateurs à choisir parmi les options suivantes :  
  1. Commencer une nouvelle partie  
  2. Afficher les scores  
  3. Quitter le jeu  

  Si l'utilisateur choisit de commencer une nouvelle partie, une grille vide est affichée,  
  et les joueurs sont invités à entrer leurs coups en choisissant la ligne et la colonne où placer leur symbole. Les joueurs jouent à tour de rôle jusqu'à ce qu'un gagnant soit déterminé ou que la grille soit remplie.  
  Si l'utilisateur choisit d'afficher le score, le programme montre les scores cumulés des parties précédentes (victoires, défaites, égalités).  
  Si l'utilisateur choisit de quitter, le programme se termine.  
  
  
Règles :  
  1. Le joueur 'X' commence toujours la partie.  
  2. Les joueurs doivent indiquer une case vide pour jouer.  
  3. Le jeu se termine lorsqu'un joueur aligne trois symboles identiques ou lorsque toutes les cases sont occupées.  
  4. Système de points :  
     - Si un joueur gagne, il marque 2 points dans son score.  
     - En cas d'égalité, les deux joueurs gagnent 1 point chacun.
